# 📊 Samsung Supply Chain & Logistics Dashboard

## 📌 Project Overview

This project is an end-to-end **Supply Chain & Logistics Analytics Dashboard** built using Power BI. It provides a comprehensive analysis of business operations including supplier performance, inventory management, shipment tracking, and customer behavior.

The dashboard transforms raw data into actionable insights to support **data-driven decision-making** in supply chain operations.

---

## 🚀 Key Features

* 📦 Supplier performance analysis (cost, lead time, reliability)
* 🏭 Inventory & production tracking
* 🚚 Shipment trends and delivery performance
* 👥 Customer behavior and sales insights
* 📊 Multi-page interactive dashboard with navigation
* 🎯 KPI-driven analytics for business monitoring

---

## 📊 Key Performance Indicators (KPIs)
